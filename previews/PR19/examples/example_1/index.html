Example 1

In this example, we solve the transit linear chemo-mechanical problem in FE² framework that is discussed the documentation in a domain with particles ($\Omega^\text{P}$) embedded in matrix ($\Omega^\text{M}$).

The full program, without comments, can be found in the next section.

using YiyuanStudentProject

Fine scale

Setting up the dimension, material parameters, loadcase originated from macro scale for solving a 3D RVE problem. The material parameters are characterized using simple values for the particle P and Matrix M. Parameters like the average particle diameter d, the particle volume fraction ϕ, and the meshsize are given followed.

dim = 3
